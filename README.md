**Descrição**

Aprender a analisar a complexidade temporal de algoritmos utilizando notação Big-O.

**Plano de ação**

- Estudar o conteúdo apresentado nas aulas.

- Analisar a complexidade de algoritmos simples implementados em código.

# 5. Plano de uso de IA

Pretendo utilizar ferramentas de inteligência artificial como apoio no processo de aprendizagem, principalmente para esclarecer dúvidas conceituais e compreender diferentes abordagens para resolução de problemas.

A IA será utilizada principalmente para:

- Explicar conceitos difíceis.
- Sugerir exemplos adicionais.
- Revisar soluções após eu tentar resolver exercícios por conta própria.

Para evitar dependência excessiva, estabeleço os seguintes limites:

- Sempre tentar resolver exercícios antes de consultar IA.
- Utilizar a IA como ferramenta de apoio e não como substituição do raciocínio.
- Revisar e compreender qualquer solução sugerida.

Dessa forma, a IA será utilizada como instrumento de aprendizado e não como atalho para respostas automáticas.
